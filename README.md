# Etsy POD 趋势分析工具

一个现代化的 Web 应用，用于分析 Etsy 上 POD（Print on Demand）产品的市场趋势数据。

## 功能特点

### 📊 仪表板
- 热门品类概览
- 流行风格趋势
- 季节性热点
- 新兴利基市场发现
- 趋势变化图表
- 品类份额分布

### 📦 品类分析
- 服装类（T恤、卫衣、帽子）
- 家居装饰（海报、帆布画、枕套）
- 配件类（马克杯、手机壳、帆布包）
- 贴纸文具
- 子品类详细趋势

### 🎨 风格趋势
- 热门设计风格标签云
- 流行色彩趋势
- 风格详细分析

### 📅 季节趋势
- 年度季节性热点时间线
- 即将到来的节日/事件
- 季节性销售趋势图表

## 快速开始

### 方法 1：直接打开
1. 双击 `index.html` 文件
2. 浏览器会自动打开应用

### 方法 2：使用本地服务器
如果需要完整功能，可以使用简单的 HTTP 服务器：

```bash
# Python 3
python -m http.server 8080

# 然后访问 http://localhost:8080
```

## 技术栈

- HTML5
- CSS3（自定义深色主题）
- JavaScript（原生 ES6+）
- Chart.js（数据可视化）
- Google Fonts（Outfit + Noto Sans SC）

## 文件结构

```
├── index.html      # 主页面
├── styles.css      # 样式文件
├── data.js         # 趋势数据
├── app.js          # 应用逻辑
└── README.md       # 说明文档
```

## 数据说明

应用使用模拟数据展示功能，包括：
- 8 个主要 POD 品类
- 10 种设计风格
- 12 种流行颜色
- 12 个月季节性数据

## 自定义数据

编辑 `data.js` 文件可以更新：
- 品类信息和增长数据
- 设计风格和描述
- 颜色趋势
- 季节性事件
