## 功能模块

### 仪表板
- 4 个统计卡片（热门品类、流行风格、季节热点、新兴利基）
- 趋势变化折线图
- 品类份额饼图
- 3 个排行榜（热门品类、流行风格、增长最快）

### 品类分析
- 4 大 POD 品类卡片（服装、家居、配件、贴纸）

### 风格趋势

## 部署流程

### 第一步：代码推送到 GitHub

```bash
cd /Users/donnaliu/Desktop/Liu/cursor/1
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/GivingGetting/etsy-pod-trends.git
git push -u origin main
```

### 第二步：Vercel 部署

1. 打开 https://vercel.com，使用 GitHub 账户登录（**无需信用卡**）
2. 点击 "Add New Project"
3. 选择 `etsy-pod-trends` 仓库
4. Framework Preset 选择 "Other"
5. 点击 "Deploy"，约 30 秒完成

> Vercel 免费计划包含：100GB/月带宽、无限部署次数、自定义域名、免费 SSL、全球 CDN。

### 第三步：绑定自定义域名

**方式 A（推荐）：修改 Nameservers**
在 GoDaddy 中将 Nameservers 改为：
```
ns1.vercel-dns.com
ns2.vercel-dns.com
```

**方式 B：手动添加 DNS 记录**
| 类型 | 主机记录 | 记录值 |
|------|----------|--------|
| A | @ | 76.76.21.21 |
| CNAME | www | cname.vercel-dns.com |

> DNS 生效时间：通常 5-30 分钟，最长 48 小时。
